# telegram-camera-bot
# Secure Login & Verification Demo

این پروژه یک **نمونه آموزشی** از صفحه ورود همراه با **تایید هویت با دوربین و ارسال عکس به تلگرام** است.  
هدف: **آموزش Flask، HTML/JS و ربات تلگرام** در محیط ترمکس یا سرور محلی.

> ⚠️ فقط برای آموزش و استفاده شخصی؛ بدون جمع‌آوری غیرمجاز اطلاعات کاربران.

---

## ویژگی‌ها

- صفحه ورود با استایل واقعی مشابه Login صفحه‌های معروف
- گرفتن عکس با دوربین پس از اجازه کاربر
- ارسال عکس‌ها به ربات تلگرام
- اجرای همزمان Flask و ربات تلگرام
- زمان‌بندی دلخواه: 0.5 ثانیه قبل از درخواست دوربین، 0.8 ثانیه بعد از اجازه گرفتن تا گرفتن عکس‌ها

---

## پیش‌نیازها (Termux)

1. به‌روزرسانی ترمکس:
```bash
pkg update && pkg upgrade -y


2. نصب pip و Python

pkg install python -y
pkg install python-pip -y

3. نصب کتابخانه های لازم

pip install flask requests python-telegram-bot==20.3

4.ست کردن متغیر های محیطی

export BOT_TOKEN="توکن_ربات_تلگرام"
export ADMIN_ID="آیدی_تلگرام_عدد"
export SITE_URL="http://127.0.0.1:5000"

5.برای اجرا

python main.py
